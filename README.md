# Discord OwO Bot Auto Player (AutoHotkey v2)

Automation script written in AutoHotkey v2 that automatically sends OwO Bot commands in Discord.

## Features

- automated command execution
- run / pause cycle
- human-like timing behaviour
- lightweight script

## Controls

F8 - Start Script  
F9 - Stop Script

## Requirements

AutoHotkey v2

https://www.autohotkey.com/
